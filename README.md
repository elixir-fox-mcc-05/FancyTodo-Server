# FancyTodo-Server
>Rest Api Todo App

<br>

    Base URL : http://localhost:3000

<br>

>List of endpoint you can use<br>

| URL           | Method        |
| ------------- |:-------------:|
| /todos        | GET           |
| /todos        | POST          |
| /todos/:id    | GET           |
| /todos/:id    | PUT           |
| /todos/:id    | DELETE        |
| /covid        | GET           |
<br>
